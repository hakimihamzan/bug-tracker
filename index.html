Introduction
Welcome to the Bug Tracker. Thank you for visiting our website and help keeping us on our toes. The purpose of this website is to give users platform to submit any software defects they encountered while using our applications. If you intend to submit bugs or issues:
  1. Please login with Google
  2. Click on submit a bug
  3. Enter your title and/or description of the bug
  4. You can suggest a Github repo to help solve the bug. Click on the plus button and search your repo. Then add it to repo list
  5. Hit submit
All submitted data will be sent to our database and we will squash every bug accordingly. Thank you.
Dev to Devs
This web app is a single page application that is built with Vanilla JavaScript, HTML&CSS, Firebase and GitHub API. It demonstrates my skills as a software developer. The project is undergoing refactoring with MVC, so that the spaghetti JS code is no more. I implemented Authentication and Authorization using Firebase and separates roles between users.
